# Instalar en Samsung S21 Ultra - for dummies

## Opción recomendada gratis: GitHub Pages

### En tu computador

1. Crea una cuenta en GitHub si no tienes.
2. Crea un repositorio nuevo llamado, por ejemplo:

```text
worldcup2026-mobile
```

3. Sube todos los archivos de esta carpeta al repositorio.
4. En GitHub entra a:

```text
Settings → Pages
```

5. En `Build and deployment`, elige:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

6. Guarda.

GitHub te dará una URL parecida a:

```text
https://TU_USUARIO.github.io/worldcup2026-mobile/
```

### En tu Samsung S21 Ultra

1. Abre esa URL en Chrome.
2. Toca los 3 puntos `⋮` arriba a la derecha.
3. Toca `Agregar a pantalla principal` o `Instalar app`.
4. Acepta.
5. Abre `WorldCup26` desde el ícono del celular.

## Actualizar la app

Cuando yo te entregue una nueva versión, reemplazas los archivos en GitHub. Tu celular toma la actualización automáticamente después de abrir/cerrar la app.

## Si la API falla

La app igual abre con el fixture local y cache. Los marcadores live dependen de que ESPN permita la consulta desde el navegador.
